Financial Services (FSI) - Credit Risk / Credit Decisioning

A Databricks Demos reference dataset demonstrating an end-to-end credit risk ML pipeline using the Medallion architecture (Bronze → Silver → Gold → Feature Store → ML Training). The target use case is predicting customer loan default.

⚙ Architecture Pattern

-- Lakeflow / DLT Spark Declarative Pipeline Kafka / Auto Loader | Bronze (Streaming Tables) <-- account, customer, credit_bureau, fund_trans, relationship, telco | Silver (Materialized Views) <-- dedup + cleanse | Gold (Materialized Views) <-- aggregated 360-view per customer | credit_decisioning_features <-- Feature Store (join of all Gold, 62 features) | credit_risk_train_df <-- labeled subset for ML training (13,479 rows) | ML Model: Credit Default Prediction (target: defaulted = 1 / 0)

📊 Data Domain Breakdown

Customer Demographics

customer_gold (59 cols)
  • age, gender, education, marital_status
  • is_resident, months_employment
  • months_current_address, join_date
  • passport_expiry, visa_expiry, document_id

Financials & Relationships

relationship_bronze → customer_gold
  • income_monthly, income_annual, tot_assets
  • debt_income_rt_w/wo_mortgage
  • total_cards, total_loans, total_mortgages
  • card/loan balance & limit amounts
  • overdraft_balance, total_deposits, total_equity
  • customer_revenue, revenue_12m, customer_score

Account Behavior

account_bronze → account_silver
  • num_accs (number of accounts)
  • avg_balance, balance_usd
  • available_balance_usd
  • CDC operation tracking

Credit Bureau

credit_bureau_gold (16 cols)
  • AMT_CREDIT_SUM, AMT_CREDIT_SUM_DEBT
  • AMT_CREDIT_MAX_OVERDUE, AMT_CREDIT_SUM_OVERDUE
  • CREDIT_DAY_OVERDUE, CNT_CREDIT_PROLONG
  • DAYS_CREDIT, DAYS_CREDIT_ENDDATE

Fund Transactions

fund_trans_gold (25 cols)
  • Sent txn count & amount (3m / 6m / 12m)
  • Received txn count & amount (3m / 6m / 12m)
  • Distinct payer/payee counts per window
  • Average sent/received amounts per window

Telecoms Behavior

telco_gold (10 cols)
  • phone_bill_amt, avg_phone_bill_amt_lst12mo
  • number_payment_delays_last12mo
  • pct_increase_annual_number_of_delays_last_3_year
  • is_pre_paid, callers, callees

ML Features & Target

credit_decisioning_features / credit_risk_train_df
  • 62 features combining all domains above
  • Derived: ratio_txn_amt_3m_12m, ratio_txn_amt_6m_12m
  • Aggregated totals: tot_txn_cnt/amt (3m/6m/12m)
  • Target: defaulted (1 = defaulted, 0 = not)
